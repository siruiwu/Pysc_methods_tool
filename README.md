# Method Compass MVP

## Files

- `index.html` - the single-page MVP site
- `css/styles.css` - shared design system and responsive layout
- `js/data.js` - starter content and Method Finder decision data
- `js/site.js` - rendering logic and Method Finder interaction

## How to view it

Open `index.html` in a browser.

## Current scope

- Homepage and project positioning
- Narrative Question Clarifier that starts from a student-written research question
- Research Question Quest turn-based mini-game for rescuing weak research questions through move-based narrowing decisions
- Guided Method Finder
- Method Library cards with search, filtering, worked-example links, and broader coverage for repeated, count, mediation, and moderation questions
- Compare Methods, Research Scenarios, Learning Paths, and Resources sections with curated external references
- About and FAQ sections

## Next good steps

1. Expand the starter cards into fuller page-level content.
2. Refine the Question Clarifier heuristics and add more rewrite patterns.
3. Expand the game with more rounds and discipline-specific question examples.
4. Refine the Method Finder branching for more nuanced study designs and mechanism questions.
5. Add downloadable worksheets or class-ready activities tied to the learning paths.
6. Migrate to a multi-page framework later if you want a richer content system.
